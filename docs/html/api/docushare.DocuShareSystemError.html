DocuShareSystemError

exception docushare.DocuShareSystemError(error_code, error_message, docushare, url)

Raised if the DocuShare site encounters a system error.

Parameters

  • error_code (str) – Error code returned by DocuShare.

  • error_message (str) – Error message returned by DocuShare.

  • docushare (DocuShare) – DocuShare instance to get the context for detailed error messages.

  • url (str) – URL that caused this error.
